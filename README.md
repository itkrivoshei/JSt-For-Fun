## üìÅ Repository Structure

The repository is thoughtfully organized for easy navigation and development:

- `üìÇmedia`: Contains all media files, including GIFs for each project.
- `üìÇpublic`: Holds public assets, including HTML and favicon.
- `üìÇsrc`: The source directory.
  - `üìÇassets`: All static assets like images and sounds for projects.
  - `üìÇcomponents`: Individual project directories with their components.
  - `üìÇredux`: Redux setup, including actions and reducers.

## üöÄ Getting Started

To dive into these projects:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/itkrivoshei/OdinMonoWebApps.git
   ```
2. **Navigate to the App Directory**:
   ```bash
   cd OdinMonoWebApps
   ```
3. **Install Dependencies**:
   ```bash
   npm install
   ```
4. **Launch the Projects**:
   ```bash
   npm start
   ```
